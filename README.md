# Лабораторный практикум по инфокоммуникационным системам и сетям (ИКСС)

В данном репозитории содержатся выполненные практические и лабораторные работы, посвященные изучению физического уровня компьютерных сетей, проектированию топологий и конфигурированию сетевого оборудования в эмуляторе **Cisco Packet Tracer**.

## 🚀 Реализованные темы и технологии

В ходе выполнения практикума были изучены и настроены следующие сетевые архитектуры и протоколы:

1. **Физический уровень и сетевые интерфейсы:** Изучение сред передачи данных (коаксиальный кабель, витая пара), определение параметров и настройка сетевых адаптеров в ОС, работа со спецификациями MAC-адресов.
2. **Базовая коммутация и симуляция:** Построение локальных сетей с использованием концентраторов (Hub), анализ движения пакетов данных и работы протоколов **ICMP** и **ARP** на уровнях модели OSI в режиме симуляции. Подключение и проверка работы сетевой периферии (принтеров).
3. **Маршрутизация (Routing):** Настройка сетевых интерфейсов маршрутизаторов. Развертывание механизмов **статической маршрутизации** по прямому соединению и маршрутов "по умолчанию" (default route), составление таблиц маршрутизации (ТМ).
4. **Управление через CLI и Безопасность:** Работа с интерфейсом командной строки (CLI) операционной системы Cisco IOS. Разграничение прав доступа, установка паролей (`enable password`, `enable secret`) на вход в привилегированный режим.
5. **Виртуальные локальные сети (VLAN):** Настройка управляемых коммутаторов **Cystal / Catalyst 2960**, разделение портов по виртуальным сетям (VLAN), обеспечение изоляции трафика на канальном уровне.
6. **Сетевые сервисы (Application Layer):** Построение инфраструктуры клиент-серверного взаимодействия. Конфигурирование серверов под задачи **DHCP** (динамическое распределение IP-адресов), **DNS** (разрешение доменных имен) и **HTTP/Web** (размещение и проверка доступности сайтов через браузер).

## 🛠 Используемый стек и инструменты

* **Среда моделирования:** Cisco Packet Tracer (версии 8.x)
* **Операционные системы оборудования:** Cisco IOS CLI
* **Сетевые утилиты диагностики:** `ping`, `traceroute`, `ipconfig /all`, `show ip route`, `show running-config`, `show vlan brief`

## 📂 Структура репозитория

Каждый каталог содержит файлы конфигурации проекта `.pkt` и сопроводительные материалы:
* Базовое IP-скрещивание двух узлов кроссоверным кабелем.
* Анализ широковещательного шторма в сетях с хабами и подключение общего принтера.
* Объединение нескольких подсетей через каскад роутеров со статическими маршрутами.
* Консольное администрирование коммутатора и конфигурирование `subnet 5` / `subnet 6` внутри VLAN.
* Стенд автоматического получения IP по DHCP и открытия веб-страницы `www.rambler.ru` через DNS-сервер.
